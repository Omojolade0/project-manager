# Project Manager

A full-stack project management application with user authentication.

## Project Structure

- `backend/` - FastAPI backend with JWT auth
- `frontend/` - React frontend 

## Features

- User authentication (JWT)
- Project management with CRUD operations
- Task tracking
- User data isolation

## Tech Stack

**Backend:**
- FastAPI
- SQLModel (SQLAlchemy ORM)
- SQLite
- JWT authentication with bcrypt

**Frontend:**
- React
- Tailwind CSS
- shadcn/ui

See README in `backend/` folder for backend setup instructions.
